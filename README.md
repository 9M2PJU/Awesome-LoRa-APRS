## üöÄ Projects

* [LoRa\_APRS\_iGate](https://github.com/lora-aprs/LoRa_APRS_iGate) ‚Äì Active and well-supported ESP32 iGate with OLED and GPS support.
* [LoRa\_APRS\_Tracker](https://github.com/lora-aprs/LoRa_APRS_Tracker) ‚Äì Official ESP32-based tracker firmware.
* [trackerjacker](https://github.com/merbanan/trackerjacker) ‚Äì Experimental multi-mode APRS tracker.
* [LightLoRaAPRS](https://github.com/lightaprs/LightLoRaAPRS) ‚Äì Unified firmware for LightAPRS/LightGateway boards.
* [qdp/LoRa\_APRS\_iGate\_TTN](https://github.com/qdp/LoRa_APRS_iGate_TTN) ‚Äì Gateway bridging LoRaWAN (TTN) and APRS-IS.
* [OZ1EKD/openmodem](https://github.com/OZ1EKD/openmodem) ‚Äì APRS modem firmware for Teensy, ESP32 and more.
* ‚ùå *Removed*: OE5BPA/LoRa-APRS-Gateway, dj0abr/LoRa-Modem ‚Äì repo unavailable or outdated.

* **WisBlock RAK19003** ‚Äì Alternate WisBlock base board

* **HELTEC Wireless Tracker**
* **WiFi LoRa32 V3.2** ‚Äì ESP32 with LoRa + GPS
* **WiFi LoRa32 V3** ‚Äì Earlier revision with similar features
